⚡ Your Energy — Освітній Вебпроєкт Цей репозиторій містить курсову роботу:
сучасний застосунок для моніторингу фізичної активності та ресурсів. Основна
увага при розробці приділялася архітектурній чистоті, швидкодії інтерфейсу та
зручності користувача (UX/UI).

🛠 Технологічна база Додаток розроблено з використанням сучасного
інструментарію, що забезпечує швидку розробку та ефективну збірку:

Середовище: Node.js

Інструмент збірки: Vite ⚡ (забезпечує миттєвий Hot Module Replacement)

Стек: Чистий JavaScript (ES6+)

Стилізація: Адаптивна верстка (Flexbox, Grid) із впровадженням кастомної
типографіки (DM Sans)

Управління залежностями: npm

🌟 Основні можливості та особливості Масштабованість: Завдяки модульній
структурі в папці src/, код легко підтримувати та розширювати.

Інтерактивність: Налагоджена валідація вхідних даних у формах та механізм
зворотного зв'язку через email.

Продуктивність: Оптимізована обробка шрифтів, іконок та медіафайлів для
мінімального часу завантаження.

Стандартизація: Дотримання єдиного стилю написання коду завдяки конфігурації
.editorconfig.

🚀 Початок роботи Отримайте копію проєкту:

Bash git clone https://github.com/your-username/YourEnergyCourseProject.git
Встановіть необхідні модулі:

Bash npm install Запустіть локальний сервер:

Bash npm run dev Створіть фінальну збірку:

Bash npm run build 📂 Архітектура репозиторію Короткий опис ключових директорій:

src/ — ядро проєкту (компоненти, стилі та бізнес-логіка).

css/ — стилі, включаючи глобальні змінні та модульні таблиці.

fonts/ & img/ — оптимізовані статичні ресурси для швидкого рендерингу.

js/ — функціональна частина: робота з API, обробка подій та валідація.

partials/ — фрагменти HTML-коду для створення модульної структури сторінок.

main.js — центральний вузол, де ініціалізуються всі модулі.

index.html & favorites.html — основна сторінка та розділ збережених вправ.

vite.config.js — параметри конфігурації середовища Vite.

Проєкт створено в рамках курсу з веброзробки для закріплення навичок роботи з
сучасними інструментами. 🎓💻
