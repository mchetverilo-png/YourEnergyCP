### ⚡ Your Energy — Course Project
Навчальний проєкт вебдодатку для моніторингу та керування фізичними ресурсами. Проєкт фокусується на продуктивності, чистій архітектурі та сучасному UX/UI.

### 🚀 Технологічний стек
Проєкт побудований на базі Vite, що забезпечує миттєве гаряче перезавантаження (HMR) та оптимізовану збірку.

- Runtime: Node.js

- Build Tool: Vite ⚡ (швидкість та сучасні стандарти)

- Мова: JavaScript

- Стилізація: Сучасна верстка з використанням Flexbox/Grid та кастомних шрифтів (DMSans)

- Пакетний менеджер: npm

### 🛠 Ключовий функціонал
1. Модульна структура: Весь вихідний код організовано в директорії src/, що дозволяє легко масштабувати проєкт.

2. Валідація та комунікація: Реалізовано логіку роботи з формами та функціями відправки повідомлень (email functionality).

3. Оптимізація ресурсів: Налаштована обробка статичних файлів, шрифтів та іконок для швидкого завантаження сторінок.

4. Професійна розробка: Використання .editorconfig для підтримки єдиного стилю коду та .gitignore для чистоти репозиторію.

### 📦 Встановлення та запуск
Клонуйте репозиторій:
```Bash
git clone https://github.com/your-username/YourEnergyCourseProject.git
```
Встановіть залежності:
```Bash
npm install
```
Запустіть режим розробки:
```Bash
npm run dev
```
Зберіть проєкт для продакшену:
```Bash
npm run build
```
### 📂 Огляд структури

```text
.
├── src/                — логіка додатку, компоненти та стилі.
│   ├── css/            — каскадні таблиці стилів (скидання стилів, глобальні змінні та модулі).
│   ├── fonts/          — локальні шрифти (DMSans), інтегровані для швидкого рендерингу.
│   ├── img/            — графічні активи (іконки, ілюстрації), що оптимізуються при збірці.
│   ├── js/             — основна логіка: API-запити, валідація форм та інтерактивні елементи.
│   ├── partials/       — модульні HTML-блоки для зручної та чистої верстки сторінок.
│   ├── main.js         — головна точка входу, що об'єднує всі модулі системи.
│   ├── index.html      — вхідна точка та каркас додатку, що об'єднує всі модулі.
│   ├── favorites.html  — окрема сторінка додатка, призначена для відображення та керування "обраними" вправами.
├── vite.config.js      — конфігурація збірки.
└── package.json        — керування залежностями та скриптами.
```
Розроблено як курсовий проєкт з фокусом на сучасні вебтехнології. 🎓💻
