⚡ Your Energy — Освітній Вебпроєкт Цей репозиторій містить курсову роботу:
сучасний застосунок для моніторингу фізичної активності та ресурсів. Основна
увага при розробці приділялася архітектурній чистоті, швидкодії інтерфейсу та
зручності користувача (UX/UI).

🛠 Технологічна база Додаток розроблено з використанням сучасного
інструментарію, що забезпечує швидку розробку та ефективну збірку:

Середовище: Node.js

Інструмент збірки: Vite ⚡ (забезпечує миттєвий Hot Module Replacement)

Стек: Чистий JavaScript (ES6+)

Стилізація: Адаптивна верстка (Flexbox, Grid) із впровадженням кастомної
типографіки (DM Sans)

Управління залежностями: npm

🌟 Основні можливості та особливості Масштабованість: Завдяки модульній
структурі в папці src/, код легко підтримувати та розширювати.

Інтерактивність: Налагоджена валідація вхідних даних у формах та механізм
зворотного зв'язку через email.

Продуктивність: Оптимізована обробка шрифтів, іконок та медіафайлів для
мінімального часу завантаження.

Стандартизація: Дотримання єдиного стилю написання коду завдяки конфігурації
.editorconfig. 📦 Встановлення та запуск Клонуйте репозиторій:

```Bash
git clone https://github.com/your-username/YourEnergyCourseProject.git
```

Встановіть залежності:

```Bash
npm install
```

Запустіть режим розробки:

```Bash
npm run dev
```

Зберіть проєкт для продакшену:

```Bash
npm run build
```

📂 Огляд структури

```text
.
├── src/                — логіка додатку, компоненти та стилі.
│   ├── css/            — каскадні таблиці стилів (скидання стилів, глобальні змінні та модулі).
│   ├── fonts/          — локальні шрифти (DMSans), інтегровані для швидкого рендерингу.
│   ├── img/            — графічні активи (іконки, ілюстрації), що оптимізуються при збірці.
│   ├── js/             — основна логіка: API-запити, валідація форм та інтерактивні елементи.
│   ├── partials/       — модульні HTML-блоки для зручної та чистої верстки сторінок.
│   ├── main.js         — головна точка входу, що об'єднує всі модулі системи.
│   ├── index.html      — вхідна точка та каркас додатку, що об'єднує всі модулі.
│   ├── favorites.html  — окрема сторінка відображення та керування "обраними" вправами.
├── vite.config.js      — конфігурація збірки.
└── package.json        — керування залежностями та скриптами.
```

Розроблено як курсовий проєкт з фокусом на сучасні вебтехнології. 🎓💻
